# 风车cursorfree1.1l - GUI版本使用指南

## 🎉 GUI版本特性

### ✨ 界面特点
- 🖱️ **图形化界面**: 友好的GUI界面，支持鼠标操作
- 🎨 **现代设计**: 使用Tkinter现代主题，界面美观
- 📊 **实时日志**: 实时显示操作日志和进度
- 🔄 **进度指示**: 操作过程中显示进度条
- 💬 **确认对话框**: 重要操作前会弹出确认对话框
- 🌐 **博客链接**: 界面中包含博客地址链接

### 🔧 功能按钮
1. **显示信息** - 显示Cursor编辑器和系统信息
2. **重置机器码** - 重置机器码和遥测ID
3. **清理缓存** - 清理Cursor缓存目录
4. **完整清理** - 执行所有清理操作（推荐）
5. **清空日志** - 清空操作日志显示
6. **打开备份目录** - 打开备份文件夹
7. **清理备份文件** - 删除所有备份文件（释放空间）⭐ 新增
8. **运行测试** - 运行功能测试
9. **关于** - 显示软件信息

## 🚀 安装和使用

### 自动安装（推荐）
```bash
# 进入项目目录
cd cursor_cleaner

# 运行安装脚本
./install_gui.sh
```

安装脚本会自动：
- ✅ 检查并安装Python3和tkinter
- ✅ 设置文件权限
- ✅ 创建桌面文件和应用程序菜单项
- ✅ 创建桌面快捷方式
- ✅ 创建启动脚本

### 手动安装
```bash
# 安装tkinter（如果未安装）
sudo apt install python3-tk

# 设置执行权限
chmod +x cursor_free_gui.py

# 直接运行
python3 cursor_free_gui.py
```

## 🎯 使用说明

### 基本操作流程
1. **启动程序** - 使用上述任一方式启动
2. **查看信息** - 点击"显示信息"了解当前状态
3. **选择操作** - 根据需要选择相应的清理操作
4. **确认操作** - 在弹出的确认对话框中确认
5. **查看结果** - 在日志区域查看操作结果

### 推荐操作顺序
1. 🔍 **显示信息** - 了解当前Cursor状态
2. 🧹 **完整清理** - 执行全面清理（推荐）
3. 📁 **打开备份目录** - 查看备份文件
4. 🔄 **重启Cursor** - 使更改生效
5. 🗑️ **清理备份文件** - 释放磁盘空间（可选）

### 🗑️ 清理备份文件功能详解

**功能说明**：
- 删除所有由程序创建的备份文件
- 释放磁盘空间
- 显示详细的文件统计信息

**使用场景**：
- 确认清理操作成功后，删除备份文件释放空间
- 定期清理累积的备份文件
- 磁盘空间不足时的空间回收

**安全提示**：
- ⚠️ 删除操作不可恢复
- 📊 删除前会显示文件数量和占用空间
- 💬 需要用户确认才会执行删除
- 🔒 只删除程序创建的备份文件，不影响其他文件

## 🔧 故障排除

### 常见问题

#### 1. 程序无法启动
```bash
# 检查Python和tkinter
python3 -c "import tkinter; print('OK')"

# 如果失败，安装tkinter
sudo apt install python3-tk
```

#### 2. 权限错误
```bash
# 设置执行权限
chmod +x cursor_free_gui.py
chmod +x *.sh
```

#### 3. 桌面文件不显示
```bash
# 更新桌面数据库
update-desktop-database ~/.local/share/applications/
```

#### 4. 图标不显示
- 图标文件路径可能不正确
- 可以忽略，不影响功能使用
